## Description

### Options

Le do n'accepte que les options courtes suivantes :
* -a (and)
* -o (or)
* -c (court-circuit)
* -k (kill) qui doit être utilisé avec l'option -c pour avoir un effet.


### Utilisation
	`./do {options} {commandes}`
